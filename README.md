# Timmy

Manage parallel coding agents with clear intent and durable context.

This repository hosts release artifacts for Timmy.

Source remains private for now while we finish core features and make the repo maintainable for contributors.

## Install (macOS/Linux)

```sh
curl -fsSL https://github.com/timmy-labs/timmy/releases/latest/download/install.sh | sh
```

## Install (Windows)

```powershell
irm https://github.com/timmy-labs/timmy/releases/latest/download/install.ps1 | iex
```
